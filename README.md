# Type Quest

A small, local typing-learning app for children. The first lesson introduces the home-row anchor keys **F** and **J**.

## Files

- `index.html`: the page structure and written content.
- `style.css`: the visual design.
- `app.js`: the behavior that reacts to key presses.

## Run it

For this first version, open `index.html` in a web browser. Later we will add a small local development server, automated checks, saved progress, and desktop packaging.
